# Ticket App

In the project directory, you can run:

```
  npm install
  node server.js
```
to start the backend server on http://localhost:5000

## API routes

### Tickets
- **GET /tickets/**

Gets all tickets

- **POST /tickets/**

Adds a new ticket

- **PATCH /tickets/:id**

Updates ticket status

### Tasks
- **GET /tasks/**

Gets all tasks

- **POST /tasks/**

Adds a new task

- **PATCH /tasks/:id**

Updates task status

### Actions
- **GET /actions/**

Gets all actions

- **POST /actions/**

Adds a new action

- **PATCH /actions/:id**

Updates action status
